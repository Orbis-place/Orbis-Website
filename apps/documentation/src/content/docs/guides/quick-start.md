# Quick Start Guide

Welcome to Orbis! This guide will help you get started with the platform.

## For Players

### Discovering Servers

1. Navigate to the [Servers](/servers) section
2. Use filters to find servers that match your preferences
3. Click on a server to view details and join

### Browsing the Marketplace

1. Visit the [Marketplace](/marketplace)
2. Filter by category: Mods, Plugins, Tools, or Scripts
3. Download resources to enhance your Hytale experience

## For Creators

### Publishing Content

1. Create an account and log in
2. Navigate to your dashboard
3. Click "Create New" to publish a server or resource
4. Fill in the details and submit for approval

### Managing Your Content

Access your dashboard to:
- Edit your published content
- View analytics and statistics
- Respond to user feedback
- Update versions and changelogs

## API Access

Developers can integrate with Orbis using our REST API:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.orbis.place/v1/servers
```
